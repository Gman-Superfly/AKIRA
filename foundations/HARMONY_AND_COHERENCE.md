## 2. The Comma Problem in Music

### 2.1 The Mathematical Mismatch

```
THE PYTHAGOREAN COMMA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  A perfect fifth has frequency ratio 3:2                               │
│  An octave has frequency ratio 2:1                                     │
│                                                                         │
│  If you stack 12 perfect fifths:                                       │
│  (3/2)^12 = 129.746...                                                 │
│                                                                         │
│  If you stack 7 octaves:                                                │
│  2^7 = 128                                                              │
│                                                                         │
│  THE DIFFERENCE: 129.746 / 128 ≈ 1.0136                               │
│                                                                         │
│  This is about 23.5 cents — the "Pythagorean comma"                   │
│                                                                         │
│  ═══════════════════════════════════════════════════════════════════  │
│                                                                         │
│  12 fifths SHOULD equal 7 octaves.                                    │
│  They DON'T.                                                           │
│  The circle of fifths doesn't close.                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
