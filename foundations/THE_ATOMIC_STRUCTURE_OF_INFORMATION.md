# The Atomic Structure of Information: Action Quanta (AQ)

A deep exploration of actionable, representationally irreducible information as the fundamental unit of learned representations—its structure, properties, and combinatorics.

---

## Terminology Note

**This document introduces Action Quanta (AQ)** — AKIRA's term for irreducible actionable patterns.

Williams & Beer (2010) use "information atoms" for PID decomposition terms (Redundancy, Unique, Synergy). To avoid collision with established terminology, AKIRA uses **Action Quanta (AQ)** for the quasiparticle-like patterns described in this document.

**Why "Action Quanta"?**
- Planck's constant (h-bar) IS the quantum of action in physics
- Actionability is the defining criterion for these units
- Aligns with BEC quasiparticle interpretation

**Chemistry Analogy vs State Terminology:**
- This document uses "molecules" as a **chemistry analogy** - describing how AQ combine to form concepts (like atoms → molecules)
- The AKIRA state terminology uses "crystallized" for the post-collapse state (emphasizing IRREDUCIBILITY)
- "Bonded state" is the tentative term for combined crystallized AQ (the chemistry analogy's "molecules")
- These are complementary framings: chemistry analogy describes COMBINATION, state terminology describes PHASE

See `TERMINOLOGY.md` for full terminology framework.

## 2. What Is an Action Quantum?

### 2.1 Definition

```
ACTION QUANTUM (AQ):

A unit of actionable information that is:

1. IRREDUCIBLE
   Cannot be decomposed into simpler actionable units.
   Breaking it destroys its functionality.

2. STRUCTURED
   Has internal organization (not a formless blob).
   The structure enables its function.

3. COMBINABLE
   Can join with other atoms to form compounds.
   Follows compositional rules.

4. ACTIONABLE
   Enables some action or prediction.
   Has utility, not just description.

5. CONSERVED
   Persists through transformations.
   The "same" atom in different representations.
```

### 2.2 What It Is NOT

```
ATOMIC INFORMATION IS NOT:

1. NOT a bit
   A bit is too small—it has no structure.
   Atoms are patterns, not points.

2. NOT a complete concept
   Concepts are molecules, not atoms.
   "Dog" is composed of shape + texture + behavior atoms.

3. NOT arbitrary
   Atoms are discovered, not chosen.
   They're determined by the task and data.

4. NOT context-free
   An atom's meaning depends on what it combines with.
   Like chemistry: oxygen behaves differently alone vs in water.

5. NOT infinitely divisible
   There's a floor—the irreducible level.
   Below that, you can't act.
```

## 6. Computational Irreducibility

### 6.1 Wolfram's Concept

```
COMPUTATIONAL IRREDUCIBILITY (Wolfram, 2002):

A computation is irreducible if there is NO shortcut.
You must run the full computation to know the result.

┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  REDUCIBLE:                                                     │
│  f(x) = x²                                                     │
│  To find f(1000000), you don't need to compute f(1), f(2)...   │
│  You can jump directly: 1000000² = 10¹²                        │
│                                                                 │
│  IRREDUCIBLE:                                                   │
│  Cellular automaton Rule 110                                    │
│  To find state at t=1000000, you MUST simulate all prior states│
│  No shortcut exists.                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

IRREDUCIBILITY means the system IS its own simplest description.
You cannot compress the PROCESS, only the RESULT.
```

### AKIRA's Term: Representational Irreducibility

```
REPRESENTATIONAL IRREDUCIBILITY (AKIRA)
───────────────────────────────────────

DEFINITION:
  The representation cannot be simplified further
  without losing the ability to act correctly.

THE INDUCTION:
  1. AQ are the minimum pattern for correct action
  2. If the representation could be simplified further:
     - Learning would simplify it (gradient descent, regularization)
  3. What survives training IS the minimum needed
  4. Therefore: AQ are representationally irreducible

ACTIONABLE IRREDUCIBILITY (equivalent formulation):
  Cannot reduce without losing actionability.
```
